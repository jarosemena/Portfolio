1. Optimizador de Horarios con Restricciones Múltiples

Backend (.NET): Gestión de recursos, empleados, turnos, restricciones legales
Python: Algoritmos de programación lineal (PuLP, scipy.optimize) para crear horarios óptimos
Frontend (React): Calendario drag-and-drop que muestra conflictos y optimizaciones en tiempo real

2. Sistema de Control de Inventario con Punto de Reorden Dinámico

Backend (.NET): Gestión de productos, movimientos, proveedores
Python: Modelos EOQ (Economic Order Quantity), cálculo de stock de seguridad con distribuciones de demanda
Frontend (React): Dashboard con alertas predictivas basadas en análisis estadístico de consumo

3. Calculadora Avanzada de Pricing Dinámico

Backend (.NET): Catálogo de productos, historial de ventas, competencia
Python: Análisis de elasticidad de precios, optimización de margen vs volumen, regresión múltiple
Frontend (React): Simulador interactivo que muestra curvas de demanda y punto óptimo de precio

4. Sistema de Optimización de Corte de Materiales (Cutting Stock Problem)

Backend (.NET): Gestión de órdenes, materiales, desperdicios
Python: Algoritmos de optimización para minimizar desperdicio en cortes (programación lineal entera)
Frontend (React): Visualización 2D de patrones de corte óptimos con métricas de eficiencia

5. Plataforma de Análisis de Capacidad y Cuellos de Botella

Backend (.NET): Modelado de procesos productivos, recursos, tiempos
Python: Teoría de colas, simulación de eventos discretos, análisis de capacidad con scipy
Frontend (React): Diagramas de flujo interactivos que muestran utilización y puntos críticos
