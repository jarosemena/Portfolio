# Data Directory

Esta carpeta centraliza todos los archivos JSON con datos del portfolio.

## Archivos:

### aboutme.json
Contiene la información de la sección "About Me" del portfolio.

### experiences.json
Contiene el historial de experiencia laboral con detalles de cada posición.

### skills.json
Contiene las habilidades técnicas con niveles de expertise, años de experiencia y categorías.

## Estructura:
- Todos los JSON de datos deben ubicarse en esta carpeta
- Los componentes importan desde `../../data/[archivo].json`
- Los iconos referenciados en los JSON deben estar en `src/assets/icons/`
